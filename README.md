# Music Player

- A modern player web application built with HTML, CSS, and JavaScript.

## Features

- Responsive Layout: Works on desktop and mobile devices.

- Player Controls: Play, pause, skip next/previous, repeat, and shuffle.

- Volume & Seek Control: Interactive sliders for volume and song progress.

- Customizable Styling: Easy-to-modify CSS variables.

## Technologies Used

- HTML5

- CSS3

- Google Fonts: Roboto

- Material Symbols for icons

- -JavaScript for audio playback (script.js)
